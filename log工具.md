|  方法   | 描述  |
|  ----  | ----  |
| Log.v(String String)(verbose)  | 全部信息 |
| Log.d(String String)(debug)  | 调试信息 |
| Log.i(String String)(information)  | 一般信息 |
| Log.w(String String)(warning)  | 警告信息 |
| Log.e(String String)(error)  | 错误信息 |

优先级:Error>Warning>Info>Debug>Verbose
